PRAZZAPIS

Documentation

Base Url : https://prazzapis.cyclic.app

Endpoints : GET

  • News API
    • Merdeka Trending /merdeka-trending
    • Antara News Trending /antara-news-trending
    • iNews /inews
      • iNews Trending /inews?keyword=trending
      • iNews Daerah /inews?keyword=regional
  • Anime API
    • Top Anime on Batckun /bk-top-anime
    • Anime Batch /anime-batch
  • Wallpaper API
    • Wallpaperflare /wpflare

        Parameters

      • By Search search=animal
    • WallpaperScraft /wpscraft

        Parameters

      • By Category category=animal
      • By Image Screen Size size=1280x720
  • Game API
    • Tebak Gambar /tebak-gambar
  • Utility
    • IS.GD (URL Shortener) /isgd

        Parameters

      • Url url=google.com
    • Lirik Lagu /lyric

        Parameters

      • By Title title=kokoronashi
  • APK Downloader
    • APKSFREE /apksfree

        Parameters

      • By Search /apksfree?search=[application-name]
  • Game Downloader
    • ROMSGAMES /romsgames

        Parameters

      • Game ROM /romsgames?rom=playstation-2
      • Filter by Letter /romsgames?letter=a-z/all
      • Data by Page /romsgames?page=1-99
  • E-Commerce API
    • Tokopedia /tokped

        Parameters

      • By Keyword keyword
      • Shop Tier shop_tier

        1 = Power Merchant, 2 = Power Merchant Pro, 3 = Official Store

      • Price Minimal pmin
      • Price Maximal pmax
      • Cashback cashbackm

        true or false

      • Condition condition

        1 = new, 2 = second

      • Bundling bundling

        true or false

      • Free Shipping / Gratis Ongkir free_shipping

        true or false

      • Cash on Delivery / COD cod

        true or false

      • Discount discount

        true or false

      • Grosir grosir

        true or false
